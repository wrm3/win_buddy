Use this checklist to ensure research quality:

**Objective and Scope**:

- [ ] Scope boundaries set
- [ ] Success criteria established

**Research Process**:
- [ ] Multiple sources consulted (minimum 5 for comprehensive research)
- [ ] Authoritative sources prioritized
- [ ] Sources evaluated for quality
- [ ] Information cross-verified
- [ ] Publication dates checked

**Analysis and Synthesis**:
- [ ] Findings organized logically
- [ ] Patterns identified
- [ ] Synthesis provided (not just listing findings)
- [ ] Practical implications explained
- [ ] Confidence levels noted

**Recommendations**:
- [ ] Clear recommendations provided
- [ ] Rationale explained with evidence
- [ ] Alternatives considered
- [ ] Implementation guidance included
- [ ] Risks identified

**Documentation**:
- [ ] All sources properly cited
- [ ] Executive summary written
- [ ] Professional formatting
- [ ] Gaps and limitations noted
- [ ] Next steps identified

**Quality**:
- [ ] Thorough for decision importance
- [ ] Accurate and verified
- [ ] Actionable and useful
- [ ] Well-organized
- [ ] Complete and self-contained
