# Startup Resource Access Skill

## Overview

This skill provides comprehensive guidance for accessing startup resources across ALL industries:
- Cloud infrastructure (AWS, GCP, Azure credits - $350k+)
- Lab space and equipment (BioLabs, JLABS, QB3)
- Manufacturing partners (China, Mexico, USA sourcing)
- Grant programs (SBIR, STTR, NSF - $1.8M+)
- Accelerators and incubators (YC, Techstars, industry-specific)
- University resources (tech transfer, core facilities, student talent)
- Facilities and coworking

## Quick Start

### Tech Startup
1. Apply for cloud credits (AWS + GCP + Azure)
2. Research grant programs (NSF I-Corps, NSF SBIR)
3. Consider accelerators if appropriate stage

### Biotech Startup  
1. Research lab spaces (BioLabs, JLABS, QB3)
2. Visit 3-5 facilities
3. Apply for NIH SBIR grants
4. Consider IndieBio or Rock Health accelerators

### Hardware Startup
1. Join makerspace for prototyping
2. Source manufacturers (Alibaba, Thomasnet)
3. Order samples and vet suppliers
4. Consider HAX accelerator

### Physical Product Startup
1. Define manufacturing requirements
2. Request quotes from 10+ manufacturers
3. Order samples and test quality
4. Visit factories before large orders
