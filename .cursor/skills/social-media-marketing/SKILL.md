## Platform Comparison

### Quick Reference

| Platform | Best For | Content Type | Posting Frequency |
|----------|----------|--------------|-------------------|
| YouTube | Long-form, tutorials | 8-15 min videos | 1-2x/week |
| TikTok | Discovery, viral | 15-60s vertical | 1-3x/day |
| Instagram | Visual, lifestyle | Reels, Stories | 1x/day + Stories |
| LinkedIn | B2B, professional | Text, carousels | 3-5x/week |
| Twitter/X | News, engagement | Text, threads | 3-5x/day |

### Algorithm Priorities

| Platform | Primary Signals | Secondary Signals |
|----------|-----------------|-------------------|
| YouTube | Watch time, CTR | Engagement, session time |
| TikTok | Watch %, completion | Shares, comments |
| Instagram | Saves, shares | Comments, watch time |
| LinkedIn | Dwell time, comments | Shares, reactions |

## Content Strategy

### Content Pillars Framework

```
Define 4-5 content pillars:

Example (Tech Creator):
1. Tutorials (40%) - How-to content
2. Reviews (25%) - Product/tool reviews
3. Industry News (15%) - Hot takes, trends
4. Behind the Scenes (10%) - Personal, authentic
5. Engagement (10%) - Questions, polls
```

### Content Calendar Template

```markdown
# Weekly Content Calendar

## Monday
- Platform: YouTube
- Type: Tutorial
- Topic: [Topic]
- Status: [ ] Script [ ] Film [ ] Edit [ ] Publish

## Tuesday
- Platform: TikTok/Reels
- Type: Quick tip
- Topic: [Topic]
- Status: [ ] Film [ ] Edit [ ] Publish

## Wednesday
- Platform: LinkedIn
- Type: Thread/Carousel
- Topic: [Topic]
- Status: [ ] Write [ ] Design [ ] Publish

[Continue for week...]
```

### Hook Formulas

**Opening Hooks (First 3 Seconds)**
```
Pattern Interrupt:
- "Stop scrolling - this will change everything"
- "Wait, you're doing [thing] wrong"

Curiosity Gap:
- "Nobody talks about this, but..."
- "The secret that [successful people] don't share"

Direct Value:
- "Here's how to [achieve result] in [time]"
- "3 things I wish I knew about [topic]"

Controversy:
- "Unpopular opinion: [statement]"
- "[Popular thing] is overrated. Here's why."
```

**Retention Hooks (Throughout)**
```
- "But here's where it gets interesting..."
- "Wait, it gets better..."
- "This is the part most people miss..."
- "Here's the trick..."
```

## YouTube Strategy

### Video Structure

```
0:00-0:15  HOOK (Stop the scroll)
0:15-0:45  INTRO (What they'll learn)
0:45-X:XX  CONTENT (Deliver value)
X:XX-END   CTA (Subscribe, next video)
```

### Thumbnail Best Practices

```
Elements:
✓ Face with emotion (if applicable)
✓ Bold text (3-5 words max)
✓ Contrasting colors
✓ Clear focal point
✓ Readable at small size

Dimensions: 1280 x 720 pixels

Color Psychology:
- Red: Urgency, excitement
- Yellow: Attention, energy
- Blue: Trust, calm
- Green: Growth, money
```

### YouTube SEO

```markdown
Title Formula:
[Primary Keyword] - [Benefit/Hook] | [Brand]

Example:
"Python Tutorial - Build Your First App in 30 Minutes | TechWithTim"

Description Structure:
- First 2 lines: Hook + keyword (visible before "Show more")
- Timestamps
- Links and resources
- Social links
- Hashtags (3-5)

Tags:
- Primary keyword
- Secondary keywords
- Long-tail variations
- Related topics
```

### YouTube Metrics

| Metric | Good | Excellent |
|--------|------|-----------|
| CTR | 4-6% | 8%+ |
| Avg View Duration | 40-50% | 60%+ |
| Engagement Rate | 3-5% | 7%+ |
| Sub Conversion | 1-2% | 3%+ |

## Short-Form Video (TikTok/Reels)

### Video Structure (60 seconds)

```
0-3s:   HOOK (Stop scroll)
3-10s:  SETUP (Context)
10-50s: VALUE (Main content)
50-60s: CTA + LOOP
```

### Trending Content Formats

```
1. Tutorial/How-To
   "How to [achieve result] in [time]"

2. List Format
   "5 [things] you need to know about [topic]"

3. Story Time
   "The time I [interesting experience]..."

4. Reaction/Response
   "Reacting to [viral thing]"

5. Comparison
   "[Thing A] vs [Thing B]"

6. POV
   "POV: You're a [role] doing [thing]"
```

### Posting Strategy

```
Optimal Times (test your audience):
- Morning: 7-9 AM
- Lunch: 12-2 PM
- Evening: 7-10 PM

Frequency:
- Minimum: 1x/day
- Optimal: 2-3x/day
- Maximum: Don't spam

Hashtag Strategy:
- 3-5 hashtags per video
- Mix: 1 large, 2 medium, 2 niche
```

## LinkedIn Strategy

### Profile Optimization

```markdown
Headline Formula (220 chars):
[What you do] | [Who you help] | [Credibility] | [CTA]

Example:
"Helping 10K+ creators build AI businesses | Founder @TechStartup | 
Forbes 30 Under 30 | DM me your AI questions"

About Section Structure:
1. Hook (first 2 lines visible)
2. Your story/journey
3. What you do now
4. Who you help
5. Results/proof
6. CTA
```

### Post Formats

**Text Post (Highest Engagement)**
```
[Hook - first line]

[Story or context - 3-5 lines]

[Key insight or lesson]

[Takeaway or CTA]

[Hashtags - 3-5]
```

**Carousel Post**
```
Slide 1: Hook/Title
Slides 2-9: Content (one point per slide)
Slide 10: CTA + Summary
```

**Poll**
```
[Question]

🔵 Option A
🟢 Option B
🟡 Option C
🔴 Option D

[Context or follow-up in comments]
```

### LinkedIn Algorithm Tips

```
✓ Post consistently (3-5x/week)
✓ Engage before and after posting
✓ Respond to all comments (first 2 hours critical)
✓ Use native content (no external links in post)
✓ Write for mobile (short paragraphs)

✗ Don't post and ghost
✗ Don't use too many hashtags (3-5 max)
✗ Don't share links in main post (put in comments)
✗ Don't be overly promotional
```

## Video Production

### Equipment Recommendations

**Starter ($500)**
```
- Camera: Smartphone (iPhone/Android)
- Mic: Rode VideoMicro ($60)
- Lighting: Ring light ($30)
- Tripod: Basic phone tripod ($20)
```

**Intermediate ($2,000)**
```
- Camera: Sony ZV-E10 ($700)
- Mic: Rode VideoMic Pro+ ($300)
- Lighting: Elgato Key Light ($200)
- Tripod: Manfrotto ($150)
- Teleprompter: Elgato ($200)
```

**Professional ($5,000+)**
```
- Camera: Sony A7IV ($2,500)
- Mic: Shure SM7B + interface ($500)
- Lighting: Aputure 120D ($700)
- Teleprompter: Glide Gear ($300)
- Editing: M2 Mac ($2,000+)
```

### Editing Software

| Software | Platform | Cost | Best For |
|----------|----------|------|----------|
| CapCut | Mobile/Desktop | Free | Short-form |
| DaVinci Resolve | All | Free | Professional |
| Final Cut Pro | Mac | $300 | YouTube |
| Premiere Pro | All | $23/mo | Professional |
| Descript | All | $15/mo | Podcast/talking head |

### Editing Workflow

```
1. Import & Organize
   - Create project structure
   - Import all footage
   - Sync audio if separate

2. Assembly Cut
   - Rough cut of all content
   - Remove obvious mistakes
   - Arrange in order

3. Fine Cut
   - Tighten pacing
   - Add jump cuts
   - Remove filler words

4. Enhancement
   - Add B-roll
   - Add text/graphics
   - Add music/SFX

5. Color & Audio
   - Color correction
   - Audio leveling
   - Noise reduction

6. Export
   - YouTube: 1080p/4K, H.264
   - TikTok: 1080x1920, H.264
   - Check audio levels (-14 LUFS)
```
