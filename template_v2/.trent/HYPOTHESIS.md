# HYPOTHESIS.md
## Research Project Hypothesis Tracker

This file tracks scientific hypotheses for research-type projects. It is the
parallel to `TASKS.md` for research projects — hypotheses drive experiments
the way tasks drive deliverables.

**Project Type**: research *(set during @trent-setup)*

---

## How to Use This File

1. **Capture hypotheses early** — even rough ones. "We believe X because Y."
2. **Assign each a HYP-NNN ID** before creating an experiment phase
3. **Link experiments to hypotheses** — each phase/experiment tests ≥1 hypothesis
4. **Record outcome after experiment** — do not leave outcomes blank
5. **Never delete** — superseded or invalidated hypotheses are valuable context
